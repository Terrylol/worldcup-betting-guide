#### HTML 报告模板

HTML模板位于 `references/report_template.html`，使用 `{{占位符}}` 标记待填充位置。

**占位符映射表**：

| 占位符 | 填充内容 | 示例 |
|--------|---------|------|
| `{{date}}` | 分析日期 | 2026-06-16 |
| `{{play_type}}` | 玩法名称 | 让球胜平负 |
| `{{parlay_num}}` | 串关数 | 4 |
| `{{match_count}}` | 可选赛事数 | 4 |
| `{{match_table_rows}}` | 赛事一览表行（HTML `<tr>`） | 见下方 |
| `{{match_cards}}` | 各场POWER-6证据分析卡片（HTML，供三专家决策层引用） | 见下方 |
| `{{parlay_cards}}` | 串关方案卡片（HTML） | 见下方 |

**赛事一览表行模板**：
```html
<tr>
  <td>周一013</td>
  <td>00:00</td>
  <td><span class="team-name">西班牙</span> <span class="vs">vs</span> <span class="team-name">佛得角</span></td>
  <td><span class="handicap negative">-2</span></td>
  <td><span class="sp-value">1.46 / 4.70 / 4.32</span></td>
</tr>
```

**赛事分析卡片模板**：
```html
<div class="match-card">
  <div class="match-card-header">
    <span class="match-label">周一013 西班牙 vs 佛得角</span>
    <span class="match-tag tag-crush">碾压局</span>
  </div>
  <div class="match-card-body">
    <div class="dim-row">
      <div class="dim-label">战力鸿沟<span class="weight">×15%</span></div>
      <div class="dim-content">FIFA排名2 vs 67，S级碾压。西班牙全队身价12.2亿欧 vs 佛得角5450万欧，差两个数量级</div>
      <div class="dim-score score-high">9/10</div>
    </div>
    <!-- 重复6个维度 -->
  </div>
  <div class="match-card-footer">
    <div>
      <div class="recommendation">
        <span class="rec-direction">让球胜</span>
        <span class="rec-sp">SP 1.46</span>
      </div>
      <div class="risk-note">⚠️ 西班牙友谊赛两度让2球走盘，世界杯首轮可能收力</div>
    </div>
    <div class="total-score">
      <span class="score-num">67</span><span class="score-max">/100</span>
    </div>
  </div>
</div>
```

**赛事定性 → tag样式映射**：

| 定性 | tag class | 颜色 |
|------|-----------|------|
| 碾压局 | `tag-crush` | 红色 |
| 压制局 | `tag-suppress` | 橙色 |
| 优势局 | `tag-advantage` | 金色 |
| 胶着局 | `tag-close` | 蓝色 |
| 均势局 | `tag-even` | 紫色 |

**评分 → 分数颜色映射**：

| 评分范围 | class | 颜色 |
|---------|-------|------|
| ≥8 | `score-high` | 绿色 |
| 5~7 | `score-mid` | 金色 |
| <5 | `score-low` | 红色 |

**串关方案卡片模板**：
```html
<div class="parlay-card parlay-safe">
  <div class="parlay-header">
    <span class="parlay-title">🎯 稳妥方案</span>
    <span class="parlay-sp">22.47 <small>串关SP</small></span>
  </div>
  <div class="parlay-items">
    <div class="parlay-item">
      <span class="item-match">西班牙 vs 佛得角</span>
      <span class="item-dir">让球胜</span>
      <span class="item-sp">1.46</span>
    </div>
    <!-- 重复每场 -->
  </div>
  <div class="parlay-logic">西班牙实力碾压赢2球+概率大；比利时让1球下埃及有韧性；...</div>
  <div class="confidence">置信度：<span class="stars">⭐⭐⭐⭐</span></div>
</div>
```

**三专家方案 → 卡片样式映射**：

| 方案 | 专家 | 外层class | 主题色 |
|------|------|-----------|--------|
| 稳妥 | Dr. Karl Vogel · 量化派 | `parlay-safe` | 绿色 |
| 价值 | Johnny Liu · 盘口派 | `parlay-value` | 金色 |
| 冷门 | Mia Carter · 消息派 | `parlay-dark` | 紫色 |
